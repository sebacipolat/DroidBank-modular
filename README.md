# DroidBank-modular

A banking app demonstrates modern Android development using Hilt, Coroutines, Jetpack Compose, Room, ViewModel, MVVM architecture.dark and ligth theme, offline support.

## Features
* Home,
* Cards
* Transactions.
* Login
* Onboarding
* Error handling
